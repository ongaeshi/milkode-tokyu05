Rubyで作ったソースコード検索エンジン Milkode

ongaeshi
2012/7/29

作った動機

  • コードを読むことの重要性は年々向上
  • が、支援するためのツールは余り変わっていない・・
  • "grep" + "Web検索" = ??

もっとコードリーディングを簡単にしたい!!

Milkodeの特徴

  • 高速 (20000ファイルから1秒以内に検索)
  • 行指向 (AND検索)
  • エディタでもブラウザでも

デモ

インストール

$ gem install milkode
